La idea

¿Qué es JSON?

  • 1
    JavaScript Object Notation: un formato para escribir datos.
  • 2
    En el fondo es solo texto, igual que el contenido de los archivos que ya aprendiste a leer y escribir.
  • 3
    Lo entienden casi todos los lenguajes y programas, por eso sirve para intercambiar información.
¿Para qué sirve?

Lo usas todos los días sin darte cuenta

📱 Apps del celular
🌐 Páginas web y APIs
🎮 Guardar partidas
⚙️ Configuración de programas
Su forma

¿Cómo se ve un JSON?

{
  "nombre": "Ana",
  "edad": 20,
  "aprobado": true,
  "materias": ["Cálculo", "Programación"]
}
  • "
    Las claves van entre comillas.
  • :
    Cada clave tiene un valor, separados por dos puntos.
  • ,
    Los pares se separan con comas.
Contenido

¿Qué valores puede guardar?

  • ""
    Texto entre comillas — "Ana"
  • 12
    Números — 20, 3.5
  • ✓✗
    true / false y null (vacío)
  • [ ]
    Listas de valores — como las que ya conoces
  • { }
    Otros objetos adentro
La traducción

JSON y Python se entienden

"texto"→str
número→int / float
true / false→True / False
null→None
[ ... ]→list
{ ... }→dict
Un objeto JSON (entre llaves) se convierte en un diccionario de Python: pares clave → valor.
Manos a la obra

Python ya trae el módulo json

import json

No hay que instalar nada. Solo lo importamos y usamos sus funciones.

Python → JSON

Convertir a texto: json.dumps()

import json

estudiante = {
    "nombre": "Ana",
    "edad": 20,
    "aprobado": True
}

texto = json.dumps(estudiante)
print(texto)
Salida {"nombre": "Ana", "edad": 20, "aprobado": true}
dumps = "dump string". Fíjate: True de Python se escribe true en JSON.
JSON → Python

Leer ese texto: json.loads()

import json

texto = '{"nombre": "Ana", "edad": 20}'

datos = json.loads(texto)

print(datos["nombre"])
print(datos["edad"])
Salida Ana
20
loads = "load string". Ahora datos es un diccionario y accedemos con la clave.
Con archivos

Guardar en un archivo: json.dump()

import json

estudiante = {"nombre": "Ana", "edad": 20}

with open("datos.json", "w") as archivo:
    json.dump(estudiante, archivo)
Resultado Se crea el archivo datos.json
con la información adentro.
Usamos with open(...), igual que cuando aprendiste a escribir archivos. dump (sin "s") escribe directo al archivo.
Con archivos

Leer desde un archivo: json.load()

import json

with open("datos.json", "r") as archivo:
    datos = json.load(archivo)

print(datos)
Salida {'nombre': 'Ana', 'edad': 20}
load (sin "s") lee el archivo y lo convierte en datos de Python.
Para no confundirse

Truco para recordarlas

🔤
Con s → trabaja con texto (string)
dumps · loads
📄
Sin s → trabaja con archivos (file)
dump · load
↔️
dump = guardar / convertir  ·  load = leer / recuperar
